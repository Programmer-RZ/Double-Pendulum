# Download

**Compiling from source**

- Make sure you have SFML linked statically

- Clone this repo
```
git clone https://github.com/Programmer-RZ/Double-Pendulum.git
```
- Go to the directory

- Open .sln file in Visual Studio
```
start DoublePendulumSimulation.sln
```
- Build the project and run the .exe

**Releases**

- Download the latest release for the most stable version
